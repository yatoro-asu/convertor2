# Конвертер валют

Данный скрипт представляет собой простой и удобный инструмент для конвертации валют в реальном времени. Он использует API сервиса exchangerate-api.com для получения актуальных курсов обмена между любыми двумя валютами.

Скрипт принимает на вход базовую валюту, целевую валюту и сумму для конвертации, а затем выводит результат в указанной целевой валюте.

## Как установить

1. Установка зависимостей

Для работы скрипта требуются следующие библиотеки Python:

* requests — для выполнения HTTP-запросов к API.
* python-dotenv — для загрузки переменных окружения из файла `.env`.
* argparse — для парсинга аргументов командной строки (встроенная библиотека, устанавливать не нужно).

Установите все зависимости с помощью `pip`:

```
pip install -r requirements.txt
```

2. Получение API ключа
Вам понадобится API ключ от сервиса [exchangerate-api.com](https://www.exchangerate-api.com/). 
Зарегистрируйтесь на сайте и получите бесплатный ключ (доступен ограниченный тариф).

3. Настройка переменных окружения
После получения ключа, создайте файл `.env` в корне проекта и добавьте в него строку: 

```
EXCHANGE_API_KEY=ваш_ключ_здесь
```

## Пример запуска скрипта

1. Скрипт можно запустить из командной строки с различными параметрами.

Базовый запуск (по умолчанию: RUB → USD, сумма 1.0):

```
python currency_converter.py
```

2. Указание параметров через аргументы

```
python currency_converter.py --base EUR --target USD --amount 100 
```
