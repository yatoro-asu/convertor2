Конвертер валют
Описание проекта
Данный проект представляет собой простой консольный инструмент для конвертации валют с использованием внешнего API. Скрипт позволяет пользователю указать базовую и целевую валюты, а также сумму для конвертации, и выводит результат в указанной целевой валюте.

Проект использует API сервиса ExchangeRate-API.com для получения актуальных курсов обмена.

Как установить
Вам понадобится API ключ для этого проекта — это EXCHANGE_API_KEY (ключ к API сервиса ExchangeRate-API).

Чтобы его получить, Вам нужно зайти на их официальный сайт, там Вам нужно зарегистрироваться и следовать инструкциям по получению этого ключа.

Вот пример такого ключа:
```fc53bae0f27d1e556c2065a4```

Когда вы его получите, положите его в файл .env в формате:
```EXCHANGE_API_KEY=ваш_ключ```

Python3 должен быть уже установлен. Затем используйте pip (или pip3, если есть конфликт с Python2) для установки зависимостей:
```pip install -r requirements.txt```

Пример запуска скрипта
Чтобы запустить скрипт на Windows и вывести результат конвертации, вот пример запуска:
```py main.py -b RUB -t USD -a 1000```

Используемые библиотеки
В проекте используются следующие Python-библиотеки:

requests — для выполнения HTTP-запросов к API.
python-dotenv — для загрузки переменных окружения из файла .env.
argparse — для парсинга аргументов командной строки.
Убедитесь, что они установлены в вашем окружении. Их можно установить через requirements.txt или вручную:
```pip install requests python-dotenv argparse```
